NUnitLite Download

The latest release of NUnitLite is version 1.0 Beta 2. The simplest way to install it is by using NuGet. You may also download the source code from our project site on Launchpad.

For information on features of each release, see the release notes.

Releases prior to 0.5 are still available from our old project page on Codeplex.

Between releases, you can download the source from Launchpad following the instructions on our Frequently Asked Questions page.
