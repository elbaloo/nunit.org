Getting Started with NUnitLite

You may download NUnitLite from our Launchpad project page or as a Nuget package.

If you download the zip file, simply extract all files to a convenient location, where you will be able to reference the nunitlite assembly from your test projects. No further installation is required. If you use the Nuget package, you should add it to each test assembly that uses nunitlite.

Your test assembly should be an exe file and should reference the nunitlite assembly. If you place a call like this in your Main() 

    new TextUI().Execute(args);
then NUnitLite will run all the tests in the test project, using any args you provided on the command line. Run with the -help option to see all available options.
