NUnit 2.4.8 - July 21, 2008

General

  • NUnit is now built using NAnt 0.86 beta 1

Framework

  • PlatformAttribute has been extended to accept the new keywords NT6, Vista and Win2008Server.

    Note: In the current release, NUnit is still unable to distintuish between Vista and Windows 2008 Server. Either of them will match all the above values. This problem will be addressed in the 2.5 release.

Bug Fixes

  • The Gui output panes were failing to use a fixed font.
  • An NUnitSettings.xml file of zero length was causing a crash.
  • Invoking the gui with /exclude:XXX, where XXX is a non-existent category, was causing all tests to be excluded.
  • Categories were not working properly on repeated tests.
  • A serious memory leak was fixed in the NUnit test runners.
  • Addins now work with the console runner.
