NUnit

This documentation covers the NUnit release, which corrects several issues found in the prior release and introduces some minor features. Where applicable, we have marked sections with the version in which a feature first appeared.

If you are new to NUnit, we suggest you begin by reading the section of this site. Those who have used earlier releases may want to begin with the section.
