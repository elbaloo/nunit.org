NUnit 2.4.2

This documentation covers the NUnit 2.4.2 release, which corrects several issues found in NUnit 2.4 and 2.4.1 and makes a few minor improvements. Except where there have been visual changes, screen shots still reflect the 2.4 release. Where applicable, we have marked sections with the version in which a feature first appeared.

If you are new to NUnit, we suggest you begin by reading the section of this site. Those who have used earlier releases may want to begin with the section.

See the for more information on this release.

All documentation is included in the release packages of NUnit. Beginning with NUnit 2.4.2, you may choose to download the documentation separately.
