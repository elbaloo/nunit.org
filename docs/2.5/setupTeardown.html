Setup and Teardown

NUnit provides a number of attributes to perform initialization before a test is run and to clean up after it.

Each test fixture may contain one - and no more than one - method marked with one of the following attributes:

  • nunit_doc_link("setup", "SetUpAttribute")
  • nunit_doc_link("teardown", "TearDownAttribute")
  • nunit_doc_link("fixtureSetup", "TestFixtureSetUpAttribute")
  • nunit_doc_link("fixtureTeardown", "TestFixtureTearDownAttribute")

In addition, you may provide one or more higher level setup and teardown fixtures that control the tests within a namespace or within the entire assembly using the nunit_doc_link("setupFixture", "SetUpFixtureAttribute")
