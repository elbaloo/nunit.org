Test Selection

NUnit a number of attributes to control which tests are executed in a partiular run.

  • nunit_doc_link("ignore", "IgnoreAttribute") marks tests as temporarily excluded from the run.
  • nunit_doc_link("explicit", "ExplicitAttribute") indicates that a test should only be run if directly selected.
  • nunit_doc_link("category", "CategoryAttribute") is used to group tests for selection or exclusion.
  • nunit_doc_link("platform", "PlatformAttribute") is used to include or exclude tests on particular platforms.
  • nunit_doc_link("culture", "CultureAttribute") indicates that a test should be run only for certain culture settings.
