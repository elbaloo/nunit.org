Test Identification

NUnit uses two key attributes to identify tests, supported by several others.

  • nunit_doc_link("testFixture", "TestFixtureAttribute" ) marks a class that contains tests.
  • nunit_doc_link("test", "TestAttribute") marks a test method.
  • nunit_doc_link("suite", "SuiteAttribute") is an older method of creating test suites, which is still used by some shops.
  • nunit_doc_link("description", "DescriptonAttribute" ) provides a description for a test.
  • nunit_doc_link("property", "PropertyAttribute" ) is used to associate arbitrary properties with a test.
