Getting Started with NUnit

If you haven't already done so, go to our page, select a version of NUnit and download it. The page contains instructions for installing on your system.

To get started using NUnit, read the page. This article demonstrates the development process with NUnit in the context of a C# banking application. Check the page for additional examples, including some in VB.Net, J# and managed C++.

Which Test Runner to use?

NUnit has two different ways to run your tests. The , nunit-console.exe, is the fastest to launch, but is not interactive. The , nunit.exe, is a Windows Forms application that allows you to work selectively with your tests and provides graphical feedback.
